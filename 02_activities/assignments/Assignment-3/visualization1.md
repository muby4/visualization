# Visualization 1: Outbreaks by Institution Setting (Bar Chart)

## Software used:

Python (Pandas and Matplotlib libraries)

## Intended audience:

Public health officials, policymakers, and healthcare administrators interested in understanding where outbreaks occur most frequently to inform resource allocation and preventative measures.

## Message conveyed:

This visualization highlights which types of healthcare institutions (e.g., Long-Term Care Homes, Retirement Homes) have the highest number of reported outbreaks, helping identify areas requiring targeted interventions.

## Design considerations:

Used a bar chart for clear comparison between categories.
Chose a light blue color for visual appeal and clarity.
Rotated x-axis labels (45 degrees) to improve readability of institution names.
Added clear axis titles and a descriptive chart title.

## Reproducibility:

The code is fully reproducible using Python and standard libraries, with the dataset and script saved together.
No manual steps required; running the script will generate the same chart.

## Accessibility:

High contrast between bars and background for visibility.
Clear labels and titles assist screen readers.
Chose colorblind-friendly blue shade.

## Impacted communities:

Residents and staff of healthcare institutions in Toronto, as well as healthcare workers and policy decision-makers.

## Feature selection:

Focused on Outbreak Setting because it clearly categorizes the type of institution and allows straightforward aggregation.
Excluded less relevant columns (like addresses or outbreak dates) to keep the visualization simple and focused.

## Underwater labour:

Cleaning and verifying dataset paths, exploring the dataset structure, selecting relevant columns, and troubleshooting plotting code.
