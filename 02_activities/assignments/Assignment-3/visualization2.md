# Visualization 2: Types of Outbreaks by Institution Setting (Stacked Bar Chart)

## Software used:

Tableau Public

## Intended audience:

Public health planners, epidemiologists, and city officials who need to understand the types of outbreaks affecting different healthcare institutions.

## Message conveyed:

This chart shows not only which settings have more outbreaks, but also what kinds (respiratory vs gastroenteric) are more prevalent in each setting.

## Design considerations:

Chose a stacked bar chart to show both total and component values in a single chart.
Used color to distinguish outbreak types clearly.
Simplified labels and used a descriptive title for accessibility.
Used Tableau’s built-in colorblind-friendly palette.

## Reproducibility:

The data used is exported from Python and available as a CSV.
The visualization can be rebuilt by reloading the data into Tableau and applying the same chart logic.

## Accessibility:

High contrast colors used.
Tooltips and legends improve navigation and interpretation.
A downloadable static image was included for non-interactive formats.

## Impacted communities:

Older adults in retirement homes and long-term care facilities, as well as staff who manage infection prevention.

## Feature selection:

Focused only on Outbreak Setting and Type of Outbreak to keep the visualization clear and directly tied to the key message.
Did not include address-level detail or causative agents for simplicity.

## Underwater labour:

Data exploration, understanding categorical relationships, transforming data for Tableau, and choosing effective visual encoding to balance clarity and completeness.
